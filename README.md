# Yatok Automobiles – Static Site

Ghana's #1 destination for quality premium vehicles. This is a fully static HTML/CSS/JavaScript website ready to publish on **GitHub Pages**.

## 🚀 Deploy to GitHub Pages

1. **Create a new repository** on GitHub (e.g., `yatok-automobiles`)
2. **Upload all files** keeping the folder structure:
   ```
   index.html
   assets/
     css/
       main.css
     js/
       main.js
   ```
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Choose `main` branch and `/ (root)` folder
6. Click **Save**
7. Your site will be live at: `https://your-username.github.io/yatok-automobiles/`

## 📁 File Structure

```
yatok-automobiles/

├── assets/
```

## ✨ Features

- **Hero slider** with 4 slides and auto-play
- **Featured Cars** horizontal slider
- **New Arrivals** grid
- **Brand filter** pills
- **Add to Cart** with localStorage persistence
- **Wishlist** toggle with localStorage
- **Contact form** with validation
- **Newsletter** subscription
- **Scroll reveal** animations
- **Back to top** button
- **Mobile responsive** with hamburger menu
- **Toast notifications** for all actions

## 🎨 Tech Stack

- Pure HTML5 + CSS3 + Vanilla JavaScript
- No frameworks, no build tools needed
- Google Fonts (Bebas Neue, Barlow)
- Unsplash images (no API key needed)
- `localStorage` for cart/wishlist state
